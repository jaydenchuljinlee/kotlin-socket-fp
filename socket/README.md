# Kotlin Socket Functional Programming

이 프로젝트는 Kotlin을 사용한 소켓 프로그래밍과 함수형 프로그래밍을 학습하기 위한 예제 프로젝트입니다.

## 기술 스택

- Kotlin 1.9.25
- Spring Boot 3.2.3
- Arrow-kt 1.2.0
- Redisson 3.27.2

## 시작하기

프로젝트를 실행하기 위해서는 다음 요구사항이 필요합니다:

- JDK 21
- Gradle

프로젝트를 빌드하고 실행하려면:

```bash
./gradlew build
./gradlew bootRun
```
